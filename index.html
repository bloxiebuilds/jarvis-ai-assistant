How to Use

This page simulates the experience of using the Jarvis AI Assistant. In a real setup, the microphone button would connect to ElevenLabs for voice processing, which would then trigger local PC commands via n8n.

Try it out:

Click the microphone button or select a command from the list below to see how the system processes requests.

  • "Jarvis, open Steam."
  • "I need to stream, launch OBS."
  • "Open my browser."
  • "Activate Gaming Mode."
